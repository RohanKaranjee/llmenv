## [1.0.0] - TBD

### Added
- Initial release of llmenv CLI
- Project initialization with `llmenv init`
- Global configuration management in `~/.llmenv/`
- Context building and merging from multiple layers
- Auto-detection of project context via directory traversal
- Context status display with `llmenv status`
- Project registry management with `llmenv projects`
- Profile switching with `llmenv use`
- Pin management for persistent facts
- Decision history logging per project
- AI context injection with dry-run mode
- AI API configuration for OpenAI and Claude
- Live API calls with context injection
- Interactive project switching
- Cross-platform compatibility (Mac, Linux, Windows)
- Comprehensive CLI help and documentation
- TypeScript implementation with strict mode
- Vitest testing infrastructure
- MIT License
- Complete README with usage examples
